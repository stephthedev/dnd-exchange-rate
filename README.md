# dnd-exchange-rate

A web-app calculator to optimally exchange coins in Dungeons and Dragons (5e). 
See a working calculator [here](http://stephthedev.com/dnd-exchange-rate/).

## Getting started
1. `npm install`

## Bundling js scripts
1. `grunt`

## Running tests
1. `npm test`

## Run site locally
1. `bundle install`
2. `bundle exec jekyll serve`
